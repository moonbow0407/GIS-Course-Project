## 一、总体分工

| 成员   | 负责方向       | 核心职责                                           |
| ------ | -------------- | -------------------------------------------------- |
| 成员 A | 数据与持久化   | 本地文件、文件导出、工程文件、数据库               |
| 成员 B | 地图与空间业务 | 地图交互、图层操作、查询、编辑、空间分析、界面集成 |

------

## 二、具体功能分工

| 功能模块             | 负责人   | 主要负责内容                         |
| -------------------- | -------- | ------------------------------------ |
| 矢量数据读取         | 成员 A   | Shapefile、GeoJSON、JSON 等格式读取  |
| 栅格数据读取         | 成员 A   | GeoTIFF、IMG、DEM 等栅格读取         |
| 坐标系转换           | 成员 A   | CRS 检查、重投影、坐标系异常处理     |
| 矢量数据导出         | 成员 A   | Shapefile、GeoJSON 等格式写出        |
| 栅格数据导出         | 成员 A   | GeoTIFF 等格式写出                   |
| 工程新建、打开、保存 | 成员 A   | 工程文件格式、图层配置和视图状态保存 |
| 数据库连接           | 成员 A   | PostgreSQL/PostGIS 连接与测试        |
| 数据库导入           | 成员 A   | 将图层和要素写入数据库               |
| 数据库加载           | 成员 A   | 从数据库读取图层并转换为内部图层对象 |
| 地图显示             | 成员 B   | 矢量、栅格绘制和图层叠加             |
| 地图平移缩放         | 成员 B   | 鼠标拖动、滚轮缩放、全图显示         |
| 图层面板             | 成员 B   | 显隐、激活、排序、删除、搜索         |
| 属性表               | 成员 B   | 属性查看、滚动、结果展示             |
| 点选查询             | 成员 B   | 地图点击、要素高亮、属性展示         |
| 框选查询             | 成员 B   | 矩形框选、批量选择                   |
| 属性查询             | 成员 B   | 查询条件对话框和属性筛选             |
| 新增要素             | 成员 B   | 点、线、面绘制和属性录入             |
| 删除要素             | 成员 B   | 删除选中要素                         |
| 修改要素             | 成员 B   | 属性修改和几何修改                   |
| 线简化               | 成员 B   | Douglas-Peucker 等简化算法           |
| 线平滑               | 成员 B   | 贝塞尔曲线或样条插值                 |
| 缓冲区分析           | 成员 B   | 缓冲距离设置和结果图层生成           |
| 叠加分析             | 成员 B   | 叠加、相交或裁剪等分析               |
| 分析记录             | 成员 B   | 分析结果列表和结果显示               |
| 界面按钮接入         | 成员 B   | Ribbon、主窗口、对话框和信号连接     |
| 模块测试             | 各自负责 | 谁实现的模块由谁编写和维护测试       |
| 最终联调             | 共同完成 | 接口联调、错误修复、演示准备         |
